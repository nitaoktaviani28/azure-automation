User Flow Start / Stop Resource

1 User Membuka Dashboard
Akses melalui browser
2 Pilih Subscription
Dropdown list subscription yang tersedia
3 Load Resources & Status
GET /api/resources Menampilkan VM, AKS, App Service beserta status real-time
4 User Memilih Aksi
Start All / Stop All / Individual Resource
Start All
POST /api/all/start
Stop All
POST /api/all/stop
Individual
POST /api/{type}/{rg}/{name}
5 Backend Mengirim Command ke Azure
Azure SDK → begin_start() / begin_deallocate() / begin_stop()
6 Auto-Polling Status
Frontend melakukan polling setiap 5 detik hingga status berubah
7 Status Updated
Running / Stopped / Starting / Stopping sesuai Azure Portal
